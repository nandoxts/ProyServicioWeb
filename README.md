# ProyServicioWeb

Un proyecto de servicio web desarrollado con múltiples tecnologías.

## 📋 Descripción

Este es un proyecto web que combina frontend y backend para proporcionar un servicio integral.

## 🛠️ Tecnologías Utilizadas

El proyecto está compuesto por los siguientes lenguajes de programación:

| Lenguaje | Porcentaje |
|----------|-----------|
| CSS | 69.6% |
| HTML | 13.8% |
| C# | 12.9% |
| T-SQL | 2.8% |
| JavaScript | 0.9% |

### Detalles Técnicos

- **Frontend**: HTML, CSS y JavaScript
- **Backend**: C#
- **Base de Datos**: T-SQL

## 🚀 Inicio Rápido

### Requisitos Previos

- [.NET Framework](https://dotnet.microsoft.com/) (para C#)
- Un navegador web moderno
- SQL Server (para la base de datos)

### Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/nandoxts/ProyServicioWeb.git
```

2. Navega al directorio del proyecto:
```bash
cd ProyServicioWeb
```

3. Instala las dependencias necesarias según el lenguaje que utilices.

## 📁 Estructura del Proyecto

```
ProyServicioWeb/
├── index.html          # Archivos HTML
├── styles/             # Archivos CSS
├── scripts/            # Archivos JavaScript
├── src/                # Código backend en C#
└── database/           # Scripts de base de datos T-SQL
```
